# Video Generation Web App

Веб-приложение для генерации видео с использованием современных технологий.

## Технологии

- Next.js 16.0.1
- React
- TailwindCSS
- TypeScript

## Функционал

- Современный пользовательский интерфейс с анимированным градиентным фоном
- Адаптивный дизайн
- Интеграция с Telegram Web App
- Генерация видео по запросу пользователя

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone https://github.com/sitnov0046-cell/LIKS.git
```

2. Установить зависимости:
```bash
cd webapp
npm install
```

3. Запустить проект:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000
